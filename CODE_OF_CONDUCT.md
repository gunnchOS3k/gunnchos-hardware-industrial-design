# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in the gunnchOS3k ecosystem a harassment-free experience for everyone,
regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience,
nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our standards

**Positive:** empathy, constructive feedback, consent-first data collection, youth safety.

**Unacceptable:** harassment, doxxing, unauthorized RF transmission, collecting private payload traffic.

## Enforcement

Report concerns to the repository maintainer via GitHub issues (private contact path in SECURITY.md).
Maintainers may remove contributions that violate this code.
